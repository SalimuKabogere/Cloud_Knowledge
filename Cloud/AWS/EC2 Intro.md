This is like the CPU of the cloud, it provides processing power to run the applications
It acts as a server for running the applications

Understanding this whole concept, you need to understand virtualization
This is the abstraction of a physical machine or computer
Assume you have a powerful PC with 64 cores, 512 GB RAM, 1TB storage
 Running  a single, bare metal operating system on that machine that in the end takes less resources of the whole PC wastes its usage
 
 So what needs to be done?? 
 Get a hypervisor: This is the program that is used to run other OSs on the host or physical machine or server!
 This divides the machine into multiple VM that run independently on a single physical machine.
